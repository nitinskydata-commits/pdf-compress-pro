# PDFCompress Pro - Professional PDF Compression Platform

A complete full-stack PDF compression web application with admin dashboard, ad management, and analytics tracking.

## 🚀 Features

- **PDF Compression** - Reduce PDF file size up to 70%
- **Multiple Compression Levels** - Low, Medium, High options
- **Admin Dashboard** - Complete control over the platform
- **Ad Management** - Add, edit, and manage advertisements dynamically
- **Analytics Dashboard** - Track compressions, clicks, and impressions
- **User Management** - Secure admin login system
- **Responsive Design** - Works on all devices
- **MongoDB Database** - Store all data securely

## 📋 Prerequisites

- Node.js (v14 or higher)
- MongoDB Atlas account (free tier)
- Git
